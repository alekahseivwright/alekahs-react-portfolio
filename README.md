# Personal Portfolio Website

This is my personal portfolio website built using React. It showcases my technical skills, projects, and experience as a Software Engineering Technician student.

## Overview

The goal of this project was to create a clean, responsive, and user-friendly platform to highlight my work and provide an overview of my development experience. The site is structured to present key projects, technical skills, and background information in a clear and organized way.

## Features

- Responsive design for desktop and mobile devices  
- Project showcase section with detailed descriptions  
- Clean and structured UI focused on usability  
- Navigation between sections for a smooth user experience  

## Technologies Used

- React.js  
- JavaScript  
- HTML5 & CSS3  

## What I Learned

Through building this portfolio, I strengthened my understanding of:

- Component-based architecture in React  
- Front-end layout and responsive design  
- Structuring projects for clarity and maintainability  
- Presenting technical work in a professional format  

## Purpose

This portfolio serves as a central place to present my work, including projects in full stack development, automation, and system design. It reflects both my technical abilities and my approach to building structured and user-focused applications.
